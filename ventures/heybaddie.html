HeyBaddie

AI-powered study companion for homeschool students — personalized coaching, gamification that rewards real progress, and real-time school data via Clever.

Visit heybaddie.app →
The Build Story

My daughter is homeschooled.
I built what I wanted for her.

My daughter is homeschooled. So is my wife’s daughter. Finding study tools built for kids who learn outside the traditional classroom structure — kids who need motivation systems designed for their actual environment, not a school’s — is harder than it should be.

I built what I wanted for them. The MVP shipped on Christmas Day. True story.

What We Built

Motivation systems built for how homeschool actually works.

Personalized AI Coaching

Adapts to each student’s learning patterns. Not a generic chatbot — a companion that knows how this kid learns.

Real Gamification

Rewards actual progress, not just logging in. Confetti for finishing a unit, not for opening the app.

Clever Integration

Real-time school data integration so the system knows what’s actually being taught and where the student stands.

Parent Dashboard

Shows grades, streaks, and progress — not chat history. Parents see outcomes, not surveillance.
